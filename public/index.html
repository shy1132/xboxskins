unofficial www.xbox-skins.net replacement server

there is no frontend for this website, only the backend servers for UnleashX's downloaders

full skin downloader support with an organized & sorted skin archive (originally found here)
full preview downloader support with game previews from here
to use the skin downloader, go to system > skins, press y, and then select "Xbox-Skins.net"
to use the preview downloader, go to the game, press start, select "edit item", scroll to preview, press start, and select download online
to actually see the preview, use a skin that supports them (like UnleashRed), you can find it in the skin downloader :)
